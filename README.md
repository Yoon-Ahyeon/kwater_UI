## 코드 실행 방법

```
!git clone https://github.com/Yoon-Ahyeon/kwater_UI
```

1. React 실행 방법

```
cd kwater_UI
npm install
npm run dev
```

2. Flask 실행 방법
```
cd kwater_UI
pip install flask
flask run
```
